## Description
Display PDPA consent form for first-time users. No consent = logout with explanation. Consent recorded with timestamp + version in user record and audit log.

## Acceptance Criteria
- First login -> consent form before main page
- No consent -> logout + explanation message
- Already consented -> skip consent form on subsequent logins
