## Description
Implement HMAC-SHA256 audit logging with chain integrity. Every user action creates a signed log entry. Each entry signs content + previous HMAC to create tamper-evident chain. Includes verification endpoint.

## Acceptance Criteria
- Any user action -> AuditLog entry with HMAC-SHA256 signature
- Each entry chains previous HMAC
- Verify endpoint detects tampered entries
- Failed auth attempts logged
- Wiki modifications logged
- Chat query metadata logged (privacy-safe)
