Multi-step setup wizard (5 steps) for IT Admin to create workspace, configure domain-wide delegation, create departments, and complete initial setup. State persisted in localStorage.

## Acceptance Criteria

- Back button preserves data
- Wizard complete -> workspace ready + redirect to dashboard
- Browser close + reopen -> wizard resumes from last step
