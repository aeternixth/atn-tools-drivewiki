## Description
Implement Google OAuth 2.0 SSO login flow using Passport.js google-oauth2 strategy. Users click "Login with Google" -> redirect to consent screen -> callback creates session.

## Acceptance Criteria
- User clicks login -> Google consent screen appears
- Valid OAuth callback -> JWT session created + redirect to main page
- Non-registered domain user -> error "Account not in registered organization"
- Refresh token stored encrypted in Cloud SQL
